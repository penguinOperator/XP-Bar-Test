# XP Bar Test

A simple XP bar system built using HTML, CSS, and JavaScript.  


## Includes

- 🎯 XP progress bar with smooth animation
- 🧠 Add XP by clicking a button
- 🚫 Stops at max XP (100)
- ✨ Random quote and color effect per click

## Languages Used

- HTML
- CSS
- JavaScript
